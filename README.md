# SimpleRunnerFinal — Пиксельный платформер

Мобильное приложение на React Native (Expo). Игра в горизонтальном режиме с вибрацией, сбором монет и рекордами.

## Функции
- Горизонтальная ориентация
- Пиксельная графика
- Вибрация
- Локальное сохранение рекордов
- Нет рекламы, нет сбора данных

## Требования
- Expo CLI
- iOS/Android устройство
